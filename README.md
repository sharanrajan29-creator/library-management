# LIBRARY-MANAGEMENT-SYSTEM
## Features

### 1. Add New Book
- Book ID is assigned automatically.
- Enter the **Book Title**, **Author Name**, and **Quantity**.
- The system stores the new book in the library records.

### 2. Update Book Details
- Update the **Author Name** or **Quantity** of an existing book.
- save the updated book information library record

### 3. Remove Book
- Remove a book from the library using its **Book ID** **book name**
- permanently delete the selected book record from the library

### 4. Search Book
- Search for a book using its **Book ID** or **Book Title**.
- display the complete details of the book if the record exists

### 5. View All Books
- Display all books with their **Book ID**, **Title**, **Author**, **Quantity**, and **Availability Status**.

### 6. Issue Book to User
- Issue the book to a user using the **book ID**.
- The available quantity is updated automatically.

### 7. Return Book
- Return an issued book using the **Book ID**
- to get **user id**, **user name**and update the book detail.
- update the book availability after the return 
- maintain the accurate issue and return records.
- 
### 8. View Issued Books
- display all currently issued book.
- show details of issued books and associated user information.
- 
### 9. Save Data to Files
- store all book records and issued book details in files
- ensure that data remains available even after closing the application.
- load saved data whenever the program starts.
